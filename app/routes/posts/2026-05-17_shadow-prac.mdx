## 何を作ったか

英語のシャドーイング練習に特化したブラウザベースのローカルアプリケーションを作った.

<a href="https://github.com/3kz2501/shadow-prac" target="_blank" rel="noopener noreferrer">GitHub - 3kz2501/shadow-prac</a>

YouTube動画やローカルの音声ファイルを取り込んで, 自動で文字起こし・チャンク分割し, カラオケ形式の再生と録音・自動採点ができる.

## 主な機能

### インポート
- YouTube URLまたはローカルファイルからの音声取り込み
- OpenAI Whisperによる自動文字起こし(単語レベルのタイミング付き)
- トランスクリプトを別途渡せばforced alignmentで精度向上
- 自然なポーズ位置で15-45秒程度のチャンクに自動分割

### 練習
- 単語単位でハイライトが進むカラオケ再生
- Piper TTSによる合成音声と元音声の切り替え
- `/` アノテーションで意味のまとまりごとに区切って練習可能(ブレーク機能)
- 再生速度(0-2.0x), 音量(最大200%, Web Audio GainNodeによるブースト)

### 採点
- **Accuracy(WER)**: jiwer を使った単語誤り率
- **Prosody(タイミング)**: 単語レベルのタイミング比較によるプロソディスコア
- 合成音声モードなら合成音声のタイミング, 元音声モードなら元音声のタイミングが基準になる

### 語彙
- コンテンツ内の単語を自動抽出
- wordfreqによるCEFRレベル(A1-C1+)推定
- EJDict-handによる和訳辞書(約45,000語)
- IPA発音表記

## 技術構成
- **Backend**: Python, FastAPI, SQLite, OpenAI Whisper, piper-tts, jiwer, yt-dlp
- **Frontend**: React 19, TypeScript, Vite, React Router
- **Audio**: Web Audio API(録音 + GainNode), ffmpeg(サーバー側の音声処理)
- **Deploy**: Docker Compose対応, `./start.sh` で一発起動

処理はすべてローカルで完結する. YouTubeダウンロード以外に外部APIへのアクセスはない.

## 作った背景

先日 IIW(Internet Identity Workshop) #42 に参加してきた.
セッション中の議論内容自体はなんとなく追えていたし, 知識面で劣っているとは感じなかったが, ディスカッションで即座に英語で応答するということが全然できなかった.
知っている, 理解しているのに言語の出力速度だけで負けている感覚がかなり悔しくて, 帰ってからすぐにこのツールを作り始めた.

シャドーイングは聞いた音声を即座にリピートする練習法で, リスニングとスピーキングの両方を鍛えられる.
既存のアプリもいくつかあるが, 自分の欲しい粒度での採点や, セッション音声のような長い素材を適切に分割して繰り返し練習するワークフローが欲しかったので自作した.

## なぜ自分専用のツールなのか

このツールをあえて使う理由は, 自分の関心領域に近い音声素材で練習できることにある.
汎用的なシャドーイング教材だとニュースや日常会話が多いが, 自分が実際に使いたい語彙や表現は技術系のディスカッションに偏っている.
自分はIIWのセッションをボイスメモで録音していたので, 実際に参加したセッションの内容でそのまま練習している.
最近は他の技術系カンファレンスのセッション音声なども素材として使っている.

また, シャドーイングは多くの素材を広く浅くやるよりも, 1つの素材を決めて何度も繰り返すほうが定着しやすいとされている.
このツールではセッション単位・チャンク単位で繰り返し練習できるようにしているのはそういった理由もある.
手を広げすぎず, 1つのセッションをしっかり身体に染み込ませてから次に行くのがよさそう.

## 注意事項

このツールはローカルホストで起動する完全に個人利用を前提としたものであり, サービスとして公開する予定はない.
YouTube音声の取り込み機能(yt-dlp)を含んでいるため, 利用にあたってはYouTubeの利用規約および各国の著作権法を各自で確認した上で自己責任で使用すること.
個人の学習目的での私的利用の範囲内で利用されることを想定している.
