## 詳細

## 概要

JPCERT/CCは、2025年11月のマイクロソフト月例セキュリティ更新プログラム（Patch Tuesday）に関する注意喚起を公開しました。

## 背景

マイクロソフトは毎月第2火曜日（日本時間では第2水曜日）に定例のセキュリティ更新プログラムをリリースしています。本注意喚起は、2025年11月分の更新プログラムに含まれるセキュリティ修正に関するものです。

## 対象

- Windowsオペレーティングシステム各バージョン

## 影響範囲

更新プログラムを適用しない場合、脆弱性を悪用した攻撃により、リモートからの任意のコード実行、権限昇格、情報漏洩、サービス拒否（DoS）などの被害を受ける可能性があります。特に重要度が「緊急」または「重要」と評価されている脆弱性については、早急な対応が求められます。


### 影響を受ける製品・サービス

- Microsoft Windows

- Microsoft Edge
- その他マイクロソフト製品


## 対応方法

## 推奨される対応

1. **速やかな更新プログラムの適用**
   - Windows Updateを通じて最新のセキュリティ更新プログラムを適用してください
   - Microsoft Update Catalogから手動でダウンロードすることも可能です

2. **適用前の準備**
   - 重要なデータのバックアップを取得してください
   - システムの復元ポイントを作成することを推奨します

3. **適用後の確認**
   - 更新プログラムが正常に適用されたことを確認してください
   - システムの動作に異常がないか確認してください

4. **情報の確認**
